H5 单页面如何去做适配？

共分五个主要文件

主要文件 解释
font 存放字体文件
images 存放图片
js 存放适配 rem 的 js 文件，命名为 flexible.js—我上篇文章是源代码
less 存放样式文件，分为初始化样式文件，移动端兼容性文件，自己写的样式文件
index.html

一.index.html

<!DOCTYPE html>
<!-- 改网站为中文 -->
<html lang="zh-CN">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>标题</title>
    <!-- 引入首页的css -->
    <link rel="stylesheet" href="./less/index.css" />
  </head>

  <body>
    <div class="main">
      <!-- html结构 -->
    </div>
    <!-- 引入flexible.js进行rem适配 -->
    <script src="./js/flexible.js"></script>
  </body>
</html>

3.index.less(自己的样式文件)

VScode 下Easy LESS

这里需要将前两个 less 文件导入进来，保存后插件会自动的转换成 css 文件，同时初始化 px 转 rem 定义基准值，后续的宽高根据定义的大小改变

// 字体设置 ，需下载字体放入font文件夹中
@font-face {
  font-family: 'CKTKingKong';
  /* 定义字体名称 */
  src: url('../font/HuXiaoBoNanShenTi-2.otf');
  /* IE9 Compat Modes */
  src: url('../font/HuXiaoBoNanShenTi-2.otf?#iefix') format('embedded-opentype');
  /* IE6-IE8 */
}

body {
  background-color: #dfe1e6;
  font-family: 'CKTKingKong', sans-serif;
}

// 导入初始化的base.less
@import './base.less';
// 导入移动端兼容性normalize.less
@import './normalize.less';
// px转rem定义基准值
@baseSize: 37.5rem;

// 示例
.box {
  width: (200 / @baseSize);
}
